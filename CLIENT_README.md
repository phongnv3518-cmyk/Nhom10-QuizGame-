# CLIENT MODULE

## Client Implementation
Contains client-side networking and GUI components.

### Folders:
- `client/` - Main client code
  - `auto_client.py` - Automatic client connection
  - `client.py` - Core client networking
  - `gui_client.py` - GUI client launcher
  
- `gui_client_pkg/` - Client GUI package
  - `main_window.py` - Main window interface
  - `question_panel.py` - Question display panel
  - `log_panel.py` - Log display panel
  - `event_handler.py` - Event handling logic

### Usage:
```python
# CLI client
python client/client.py

# GUI client
python client/gui_client.py
```
